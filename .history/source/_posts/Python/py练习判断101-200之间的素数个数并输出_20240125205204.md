判断素数的方法之一：用一个数分别去除以2到squrt(这个数)，如果能被整除，则不是素数。

```python
import math

h = 0
leap = 1
for m in range(101, 201):
    k = int(math.sqrt(m + 1))
    for i in range(2, k + 1):
        if m % i ==0:
            leap = 0
            break
    if leap ==1:
        print("%d" % m)
        h += 1
        if h % 10 ==0:
            print(" ")
    leap = 1
print("Total is :%d" % h)
```
